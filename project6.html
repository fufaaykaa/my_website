Project 1

"Escaping Elephant" is an endless runner, where a little sad elephant is trying to escape from the circus. This game was created as a part of the "Tools and Technologies" course in the "University of Europe for Applied Sciences". The main character's model (elephant), model of the collectibles and 3 types of obstacles, model of the circus, textures, all animations of the character and obstacles, sound effects were created by the student Taisia Malakhova. All other models, the scripts for the character control, UIs and scripts for the menus were created by the professors of UE.

3D Modeling & Texturing

  • Created 3D model of the character.
  • Designed and applied texture for the character.
  • Modeled collectibles.
  • Designed 3 types of obstacles.
  • Created textures for collectibles and obstacles.
  • Modeled and textured the circus environment (scene surrounding).

Character Animations

  • Created 7 animations for the character:
    • Idle
    • Flying
    • Jumping
    • Sliding
    • After hit (stars above the head)
    • Scared (before the run)
    • Dead

Obstacle Animations

  • Designed destruction animations for all obstacles.

Sound Design

  • Implemented elephant roar sound when hit.
  • Created destruction sound effects for obstacles.
  • Added background music.
  • Integrated elephant scream sound effect when dead.

Project 2

"Piggy Pixie" is a 3D Modeling and Animation project, which was created as a part of the "Animation for Games" course in the "University of Europe for Applied Sciences". The main character's model, texture, all animations and additional effects, such as particle systems, were created by the student Taisia Malakhova. The character is able to walk, run, jump. As the additional animations: the character has an idle animation, she reacts to the rain and the bus stop, which can be seen on the map. All the models besides Pixie, the scripts for the character control, rain and bus stop were created by the professors of UE .

3D Modeling & Texturing

  • Created the 3D model of the character.
  • Designed and applied character texture.

Character Animation

  • Created 10 animations for the character:
    • Walking
    • Idle (dancing)
    • Idle (standing)
    • Idle (head turning)
    • Jumping
    • Running
    • Watching the rain
    • Waiting for the bus 1
    • Waiting for the bus 2
    • Walking with jump
  • Set up random idle animations.
  • Created 5 layers of animation.
  • Implemented avatar mask for animation customization.

Headphones System

  • Designed headphones model.
  • Implemented activation/deactivation logic:
    • Headphones appear during the Dancing Idle Animation.
    • Headphones disappear when any other animation starts.

FX & Script-Driven Animations

  • Implemented 2 particle systems for additional effects.
  • Designed BlendTree for smooth transitions between walking and running animations.
  • Added 2 script-driven animations for rain and bus stop triggers.
